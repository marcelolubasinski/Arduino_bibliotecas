Você poderá escolher entre dois formatos para baixar uma cópia do projeto
.zip | .tar.gz 		Este projeto está distribuído em três pastas:
  • libraries onde estão localizadas as pastas de cada biblioteca, usando a mesma disposição e hierarquia da pasta de mesmo nome do arduino;
  • tutoriais onde estão localizados os tutoriais de uso das bibliotecas disponibilizadas no repositório;
  • licences onde estão localizados os textos das licenças das bibliotecas, separados em pastas com o nome da biblioteca;
Neste website você poderá conhecer melhor este trabalho, podendo inclusive participar,(sendo nesse caso necessário conhecimentos em repositórios GIT e conta no github), ou se preferir poderá entrar em contato pelo email marcelolubasinski.mobile@gmail.com. Apesar do arduino dispor de uma grande quantidade de bibliotecas, a imensa maioria está em inglês, sendo assim me comprometi a iniciar este projeto, buscando desenvolver e reunir bibliotecas para o arduino em português do Brasil (opcionalmente a documentação de algumas bibliotecas serão apresentadas tanto em português quanto em inglês, a critério do desenvolvedor). Este projeto tem um objetivo bastante grande, devido a isso quem tiver interesse em participar, considere-se convidado. Estarei postando aqui informações sobre o desenvolvimento do projeto. Obrigado pela sua atenção.
